# Job Compass 🧭

A modern job application tracking dashboard designed to help professionals manage their job search like a pipeline.

## Features

- Track job applications
- Manage interview stages
- Monitor follow-ups
- View job search analytics
- Track resume performance
- Export job logs

## Live Demo

Try the app here:

[Job Compass Live Demo] https://job-compass-app.base44.app

## Purpose

Job Compass was created to solve a simple problem:

Job searching gets chaotic when applications, follow-ups, interviews, and notes are scattered across spreadsheets and different job boards.

This app turns the job search into a structured workflow.
